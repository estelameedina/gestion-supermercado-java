# Sistema de Gestión de Supermercado

Aplicación desarrollada en Java que simula la gestión de productos en un supermercado, incluyendo control de stock, validación de datos y manejo de errores.

---

## Tecnologías utilizadas
- Java  
- Programación Orientada a Objetos (POO)  

---

## Funcionalidades
- Registro de productos con nombre, categoría, precio y stock  
- Validación de datos (precio no negativo)  
- Control de stock disponible  
- Simulación de venta de productos  
- Gestión de errores mediante excepciones personalizadas  

---

## Conceptos aplicados
- Clases abstractas (`Producto`)  
- Encapsulación  
- Uso de colecciones (ArrayList)  
- Excepciones personalizadas  
- Separación de responsabilidades  

---

## Ejemplo de uso

- Se añaden productos al sistema  
- Se intenta vender un producto con precio inválido → error controlado  
- Se simula la venta de productos con control de stock  
- Se gestionan errores cuando el producto no está disponible  

---

## Ejecución
1. Clonar el repositorio  
2. Abrir en un IDE (IntelliJ, Eclipse…)  
3. Ejecutar la clase `Main`
